# 🟡 Pac-Man

A browser-based recreation of the classic **Pac-Man** game built using **HTML**, **CSS**, and **JavaScript**. Control Pac-Man, collect pellets, avoid ghosts, and try to achieve the highest score.

## Features

* Classic Pac-Man gameplay
* Smooth player movement
* Ghost AI and movement
* Score tracking
* Responsive game interface
* Built with vanilla JavaScript (no frameworks)

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/dev-marwa510/Pac-man.git
```

2. Open the project folder.

3. Launch `index.html` in your web browser.

> No installation or additional dependencies are required.
